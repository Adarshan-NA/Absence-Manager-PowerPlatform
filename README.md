# Absence Manager

A Power Platform-based absence management solution built using Power Apps, Power Automate, and SharePoint to streamline employee leave requests, approval workflows, status tracking, and HR visibility.

## Business Problem

Many organizations still manage employee absences through emails, spreadsheets, or disconnected systems, resulting in delayed approvals, poor visibility, and manual follow-ups.

Absence Manager digitizes the entire process through workflow automation, centralized record management, and configurable approval rules.

## Key Features

* Employee absence request submission
* Manager and deputy approvals
* Automated status tracking
* Approval and rejection comments
* Configurable absence types
* SharePoint-backed data storage
* Reporting-ready architecture

## Technology Stack

* Power Apps
* Power Automate
* SharePoint Online
* Office 365 Users
* Office 365 Outlook
* Microsoft 365

## Solution Highlights

### Automated Approval Process

Requests are routed through configurable approval workflows with support for managers and deputies.

### Centralized Record Management

All requests are stored in SharePoint for visibility, auditing, and reporting.

### Configurable Absence Types

Different absence categories can have unique approval requirements and visual indicators.

### Reporting Ready

Structured data model designed for future Power BI reporting and analytics.
